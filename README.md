# Installation

 - For proper usage of mp3 you need to install ffmpeg. Tips on installation:

    * MacOS:
    Run in terminal
    ```
    ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)" < /dev/null 2> /dev/null
    ```
    brew install ffmpeg
    ```

    * Unix:
    ```
    $ sudo apt-get install ffmpeg
    ```
